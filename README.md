# Flask API with SQLAlchemy + JWT

A Flask REST API with ORM and JWT authentication.

## Features
- User registration
- Login system
- Password hashing
- JWT authentication
- Protected routes
- SQLAlchemy ORM

## Technologies Used
- Python
- Flask
- Flask-SQLAlchemy
- PyJWT
- bcrypt

## Installation

pip install flask flask-sqlalchemy pyjwt bcrypt

## Run

python app.py

## API Endpoints

POST /register  
POST /login  
GET /profile

## Purpose
focuses on ORM-based secure API development.
